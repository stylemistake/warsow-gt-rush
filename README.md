# Rush gametype for Warsow

Deathmatch with two random weapons. Score is the amount of damage you've dealt.
Gunblade frag drops health!

[Download PK3][releases]

Have fun playing!


## Building PK3

All you need is:

```
bash runnerfile.sh
```
